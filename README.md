# Rainboy fcitx 雙拼

配方： ℞ **double-pinyin-flypy**


已收錄以下雙拼方案：

  - 小鶴雙拼

## 安裝

```bash
sudo pacman -S fcitx fcitx-rime fcitx-configtool
cd ~/.config/fcitx
rm -r rime
git clone https://github.com/Rainboylvx/rime
```
